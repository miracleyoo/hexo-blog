permute

重新排列 N 维数组的维度

语法

B = permute(A,order)

说明

B = permute(A,order) 重新排列 A 的维度使其按向量 order 所指定的顺序排列。B 含有与 A 相同的值，但访问任意特定元素所需的下标的顺序已按 order 所指定的顺序重新排列。order 的所有元素都必须是唯一的正整数实数值。

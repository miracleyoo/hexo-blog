shape

tensorflow 用 3 种方式描述一个 tensor 的维数：
rank, shape, 以及 dimension number (维数)
所以 shape 和 rank 的意思的一样的，只是表达的形式不同。

rank shape dimension
0 [] 0 维
1 [D0] 1 维
2 [D0, D1] 2 维
n [D0, D1, …, Dn-1] n 维

shape 写成只包含整数的 list 或者 tuple 形式，例如 [1, 4, 2]

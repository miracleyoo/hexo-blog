Linux 查看服务器开放的端口号

Posted on 2018-08-27 Edited on 2019-10-03

问题

如何确定一台Linux主机上有哪些端口正在使用。

解决方案

  1. 安装nmap工具检测开放端口：apt-get install nmap
  2. 查看本机活跃端口：nmap 127.0.0.1

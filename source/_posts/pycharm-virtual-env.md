# 问题

如何在Pycharm中使用用于某个app的特定的虚拟环境

# 解决方案

Once a virtual environment is created, it can be added to the list of available interpreters, as a local interpreter.

## To add an existing virtual environment to the list of available interpreters

1. In the [Project Interpreter](https://www.jetbrains.com/help/pycharm-edu/project-interpreter.html) page, click ⚙ . 
2. In the drop-down list, choose  Add . 
3. In the [Select Python Interpreter](https://www.jetbrains.com/help/pycharm-edu/select-path-dialog.html) dialog box that opens, choose the desired Python executable, and click OK.
